# lazy-mirror

一键设置开发环境镜像源

## install
```bash
cargo install --git https://github.com/zcyc/lazy-mirror
```

## usage

```bash
lm set go
lm unset go
```

## help
```
A mirror setting cli for lazy

Usage: lm <COMMAND>

Commands:
  set    
  unset  
  help   Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help information
```
