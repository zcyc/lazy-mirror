# lirror [WIP]
一键设置开发环境镜像

## example
```bash
lirror set go
lirror unset go
```

## usage
```
A mirror setting cli for lazy

Usage: lazy-mirror <COMMAND>

Commands:
  set    
  unset  
  help   Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help information
```
